# 🛠️ Technology Stack

BuyOrSell is built with a modern, high-performance stack focusing on developer experience, type safety, and premium user interface.

## Core Framework
- **[Next.js 16 (App Router)](https://nextjs.org/)**: The foundation of the application, utilizing React 19 features, Server Components, and Server Actions.
- **[React 19](https://react.dev/)**: Latest React features for optimal rendering and state management.
- **[TypeScript](https://www.typescriptlang.org/)**: Strict type checking across the entire codebase for reliability.

## Styling & UI
- **[Tailwind CSS 4](https://tailwindcss.com/)**: Utility-first CSS for rapid and consistent styling.
- **[Shadcn/UI](https://ui.shadcn.com/)**: Accessible, customizable component primitives.
- **[Framer Motion / Motion 12](https://motion.dev/)**: Smooth, declarative animations for a premium feel.
- **[GSAP](https://gsap.com/)**: Complex scroll-based and timeline animations.
- **[Lucide React](https://lucide.dev/)**: Consistent and beautiful icon set.

## State & Data
- **[TanStack Query v5 (React Query)](https://tanstack.com/query/latest)**: Powerful server state management, caching, and synchronization.
- **[Zustand](https://github.com/pmndrs/zustand)**: Lightweight, performant state management for client-side global state.
- **[React Hook Form](https://react-hook-form.com/)**: Performant, flexible forms with easy validation.
- **[Zod](https://zod.dev/)**: Schema-based validation for forms and API responses.

## Utilities & Integration
- **[Axios](https://axios-http.com/)**: Centralized HTTP client for backend communication.
- **[Pino](https://getpino.io/)**: Structured, high-performance logging.
- **[i18next](https://www.i18next.com/)**: Robust internationalization framework for (EN/AR) support.
- **Google Maps API**: Deep integration for location-based search and ad posting.
- **OpenAI API**: Powering AI-assisted ad placement and semantic search.
